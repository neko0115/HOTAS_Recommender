# HOTAS Recommender（飛行模擬搖桿推薦器）

本程式提供一個 GUI（圖形介面），依照使用者的預算、匯率、偏好機種與偏好權重，推薦 1–5 組符合條件的飛行模擬操控方案，並顯示每個方案的核心評分（1–5 分）與產品說明。

---

## 功能概述

- 依預算（TWD）推薦 1–5 組方案
- 支援匯率輸入：USD / EUR / CNY → TWD
- 偏好機種：空軍 / 海軍 / 二戰 / 直升機（可多選、全選）
- 顯示核心評分（1–5 分）：
  - 機種契合
  - CP 值
  - 真機感
  - 穩定性
  - 易用性
  - 擴充性
- 預算過低或無法湊出完整 HOTAS 時，會自動切換為「僅搖桿」方案（fallback）
- 方案點選後，下方顯示該方案包含的產品清單與產品說明

---

## 系統需求

- Windows 10/11
- 若使用 Python 執行：
  - Python 3.10+（建議）

---

## 使用方式（EXE 版）

1. 執行 `HOTAS_Recommender.exe`
2. 進入「參數設定」分頁：
   - 輸入 **預算（TWD）**
   - 輸入匯率（USD/EUR/CNY → TWD）
   - 勾選偏好機種（可多選或全選）
   - 視需求調整：
     - 手搓能力（0–5）
     - 風險容忍（0–5）
     - 權重配方（穩定優先 / 真機感優先 / CP值優先 / 自訂）
3. 回到「推薦結果」分頁，按 **開始推薦**
4. 在表格中點選任一方案，下方會顯示：
   - 方案總價（TWD）
   - 子分數（1–5）
   - 方案包含的產品清單與說明

## 介面說明

### 分頁 1：推薦結果

- **開始推薦**：以目前設定計算方案並顯示推薦清單
- **推薦結果表格**：
  - 方案名稱
  - 整體（1–5）
  - 機種契合 / CP值 / 真機感 / 穩定性 / 易用性 / 擴充性
  - 總價（TWD）
- **方案 / 產品說明欄**：
  - 顯示該方案各產品的控制型式、安裝需求、價格與描述

### 分頁 2：參數設定

- **預算（TWD）**：推薦方案不得超出預算
- **偏好機種（可多選）**：影響「機種契合」分數
- **匯率（→TWD）**：將 USD/EUR/CNY 價格換算為 TWD
- **手搓能力（0–5）**：越高越能接受需要排查/改裝的產品（影響易用性）
- **風險容忍（0–5）**：越高越能接受可靠度/售後風險（影響穩定性）
- **包含「僅搖桿」方案**：
  - 勾選：推薦時允許出現僅搖桿類方案
  - 即使未勾選，若預算不足以湊出完整方案，也會自動 fallback 到僅搖桿

---

## 評分說明（1–5）

- **CP值**：在不超支前提下，越接近預算通常越高
- **真機感**：力感應 / 力回饋 / 套件與特殊功能會提高
- **穩定性**：可靠度風險與售後風險越高則分數越低（風險容忍可抵銷部分扣分）
- **易用性**：DIY 門檻、模擬器支援風險、以及安裝需求會降低分數（手搓能力可抵銷部分扣分）
- **擴充性**：可升級/可擴充/套件加成等會提高
- **機種契合**：偏好機種勾選越符合，分數越高（多選任一命中即可加分）

## 常見問題

### 1) 我預算很低，只看到「僅搖桿」方案

代表目前預算無法湊出完整 HOTAS（例如搖桿+油門），程式會自動 fallback，確保仍能給出可用的入門方案。

### 2) 匯率要填什麼？

填你要換算到 TWD 的匯率，例如：

- USD→TWD：32
- EUR→TWD：35
- CNY→TWD：4.5

### 3) 產品價格顯示 N/A

表示該產品在資料中缺少可用價格欄位（TWD/USD/EUR/CNY 任一）。
